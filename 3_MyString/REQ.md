# 과제 3

## std::string 없이 문자열 처리의 기본기를 직접 구현

이 과제에서 요구하는 핵심

1. 메모리 관리
    * 동적 할당된 char* 를 관리
    * 깊은 복사와 얕은 복사의 차이를 실습
    * 생성자/복사 생성자/소멸자를 확실히 구현하여 메모리 누수 방지

2. C 스타일 문자열 조작
    * strlen, strcpy, strcat 같은 함수를 직접 구현
    * 반복문으로 길이 세기, 복사, 붙이기, 비교 등을 직접 작성

3. 라이브러리 제한
    * <cstring>, <string>, STL 금지 → 직접 구현 필수
